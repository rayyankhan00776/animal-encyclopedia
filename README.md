# Formatting Instructions

To format your manuscript in Word and export it to .docx, follow these steps:

1. **Open Word**: Open Microsoft Word on your computer.
2. **Set Page Layout**: Ensure your document is set to A4 size and select the appropriate margins.
3. **Format Each Section**:
   - Use **Heading 1** for the title of each animal.
   - Use **Heading 2** for the required subsections.
4. **Page Breaks**: Insert a page break after each animal's section.
5. **Export**: Go to 'File' > 'Save As' and choose .docx format. Make sure to check the page settings.

Each animal should take up exactly 3 pages in the final document, with the addition of pagination control and cover page resources.
